# 📐 2.5 PolarQuant 解析 (PolarQuant Analysis)

[🏠 返回目錄](../index.md)

## 核心思想：幅度與相位的解耦 (Decoupling Magnitude and Phase)
**PolarQuant** 是 TurboQuant 的核心技術，其基本思想是利用**極座標系**來重新定義向量的表示方式。傳統的量化方法（如 INT8/FP8）通常直接對笛卡兒座標下的數值進行截斷與捨入，這在處理具有旋轉特性的向量時，容易造成嚴重的精度損失。

PolarQuant 將每個維度的數值分解為兩個獨立的部分：
1. **幅度 (Magnitude, $r$)**：代表向量在該維度上的強度或能量。
2. **相位 (Phase, $\theta$)**：代表向量的方向或旋轉角度。

## 演算法流程

1. **座標轉換**：將輸入的 $\mathbf{K}$ 與 $\mathbf{V}$ 向量從笛卡兒座標轉換至極座標。
2. **分層量化策略**：
    - **幅度量化 (Magnitude Quantization)**：由於幅度 $r$ 的動態範圍較大，我們採用較高位元的量化（例如 4-bit 或 8-bit）來保留精確的能量資訊，這對於維持 Attention Score 的準確性至關重要。
    - **相位量化 (Phase Quantization)**：相位 $\theta$ 的範圍固定在 $[0, 2\pi)$，其資訊密度較高，我們可以使用更低位元的量化（例如 2-bit 或 3-bit）來極大化壓縮空間，同時利用其角度的連續性來減少誤差。
3. **逆轉換 (De-quantization)**：在計算 Attention 時，再將量化後的 $r$ 與 $\theta$ 轉換回笛卡兒座標，以進行矩陣乘法。

## 技術優勢
- **極高的壓縮比**：透過對相位進行極低位元的量化，我們可以在不顯著犧牲模型性能的前提下，大幅降低 KV Cache 的記憶體佔用。
- **抗噪性與穩定性**：極座標下的量化對於旋轉變換具有天然的魯棒性，這使得模型在處理長文本序列時，即使在極端量化下仍能保持穩定的注意力分佈。
